# Personal Portfolio Website

## Project Description

This project is a **Responsive Personal Portfolio Website** developed as part of **Week 1 – HTML & CSS Foundations**. The website showcases personal information, skills, projects, and contact details while following modern web development best practices such as semantic HTML, responsive design, accessibility, and clean code organization.

The portfolio is designed to work seamlessly across **mobile, tablet, and desktop devices**, providing a consistent and user-friendly experience.

---

## Project Objectives

* Build a professional personal portfolio using HTML5 and CSS3
* Apply semantic HTML structure for better accessibility and SEO
* Implement responsive layouts using CSS Flexbox and Grid
* Create a mobile-friendly navigation menu
* Follow clean file organization and coding standards

---

## Features

* Fully responsive design (mobile, tablet, desktop)
* Semantic HTML5 elements (`header`, `nav`, `main`, `section`, `footer`)
* CSS Grid and Flexbox for layout management
* External and modular CSS files
* Mobile navigation with hamburger menu
* Contact form with basic validation
* Accessible markup with alt attributes and ARIA labels
* Cross-browser compatibility

---

## Technologies Used

* **HTML5** – Structure and semantic markup
* **CSS3** – Styling and responsive design
* **JavaScript** – Basic interactivity (navigation toggle)
* **Git & GitHub** – Version control and project hosting
* **VS Code** – Development environment

## Setup Instructions

Follow these steps to run the project locally:

1. Download or clone the repository
2. Open the project folder in **VS Code**
3. Install the **Live Server** extension (if not already installed)
4. Right-click on `index.html` and select **Open with Live Server**
5. View the website in your browser

## Responsive Design Details
